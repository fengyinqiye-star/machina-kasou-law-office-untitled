# Changelog

## 1.0.0 (2026-03-26)

### Features
- Next.js 14 App Router + TypeScript + Tailwind CSS
- next-intl による日英多言語対応（/ja/, /en/ ルーティング）
- トップページ（ヒーロー、強み3点、取扱業務カード、弁護士紹介、アクセス、CTA）
- 取扱業務一覧 + 6カテゴリ詳細ページ（離婚、相続、労働、企業法務、交通事故、不動産）
- 弁護士紹介ページ（仮想一郎、仮想花子）
- 事務所概要・アクセスページ（Google Maps埋め込み）
- 料金案内ページ（カテゴリ別費用テーブル）
- 相談予約フォーム（react-hook-form + zod バリデーション）
- Resend によるメール通知（オーナー通知 + 自動返信）
- レートリミット（同一IP 5分間3回）
- プライバシーポリシー・免責事項ページ
- サンクスページ（変更・キャンセル案内含む）
- framer-motion によるスクロールアニメーション
- レスポンシブ対応（375px, 768px, 1280px）
- モバイル固定フッターバー（電話 + 相談申込）
- SEO: メタタグ、hreflang、構造化データ（LegalService, Attorney, LocalBusiness）
- sitemap.xml, robots.txt 自動生成
- ダークネイビー x ゴールド x 白のデザインシステム
- Noto Serif JP + Playfair Display + Noto Sans JP フォント
